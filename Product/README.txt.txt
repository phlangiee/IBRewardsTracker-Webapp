How to run:
1. Install .NET 9.0: https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-9.0.203-windows-x64-installer
2. Open command prompt in this folder.
3. Type dotnet run into the command prompt. Hit enter.
4. Go to the link listed after "Now listening on:" in the command prompt. This should contain localhost in the URL. 

Folders/Files:
- Pages: Contains the front and backend code of each Page. Index is the login page.
- Models: Contains the code for the model classes. 
- Data: Contains the code for the functions of each model. 
- Program.cs: Contains the code for when the application first starts up.
